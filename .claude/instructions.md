# Claude 작업 규칙

이 프로젝트에서 코드를 수정하거나 업데이트할 때 반드시 따라야 할 규칙입니다.

## ⚠️ 필수 워크플로우: CLAUDE.md 관리

### 코드 변경 전
1. **반드시 `CLAUDE.md` 파일을 먼저 열고 읽을 것**
2. 현재 프로젝트 상태, 아키텍처, 알려진 이슈를 파악할 것
3. 변경하려는 내용이 기존 결정사항과 충돌하지 않는지 확인할 것

### 코드 변경 후
1. **반드시 `CLAUDE.md` 파일을 업데이트할 것**
2. 업데이트할 내용:
   - `변경 이력` 테이블에 날짜, 변경 내용, 영향 범위 추가
   - `현재 작업 중` 체크리스트 업데이트
   - 새로운 아키텍처 결정이 있으면 `핵심 결정사항`에 추가
   - 버그나 이슈 발견 시 `알려진 이슈`에 추가
3. 파일 구조가 변경되면 `아키텍처` 섹션의 트리도 업데이트

### 절대 하지 말 것
- CLAUDE.md를 읽지 않고 코드를 수정하는 것
- 코드를 변경하고 CLAUDE.md를 업데이트하지 않는 것
- CLAUDE.md의 기존 이력을 삭제하는 것

## 프로젝트 컨텍스트

- **베이스 코드**: furnishup/blueprint3d (MIT License)
- **목표**: 기존 2D 시뮬레이터의 복층(1층+2층) 방 구조를 3D로 변환한 가구 배치 앱
- **핵심 제거 사항**: 2D 평면도 편집기 (floorplanner 모듈의 UI)
- **핵심 유지 사항**: 3D 렌더링, 가구 배치/이동/회전, OBJ 모델 로딩
- **핵심 추가 사항**: 고정 구조물(화장실, 싱크대, 계단, 신발장 등)을 3D 박스로 렌더링
- **기한**: 1주일 — 완벽함보다 동작하는 결과물 우선

## 방 구조 핵심 수치 (빠른 참조용)

### 1층 (380×510cm, 벽높이 250cm)
- 화장실: 200×170, (0,0), 높이 250cm
- 싱크대: 150×60, (0,170), 높이 90cm
- 신발장: 100×70, (280,0), 높이 120cm
- 계단: 70×210, (310,225), 높이 250cm
- 뒷문: 100×6, (280,504)
- 책장: 30×100, (0,410), 높이 180cm — ⚠️ 이동 가능

### 2층 (380×480cm, 벽높이 250cm)
- 막힌공간: 215×170, (0,0), 높이 250cm
- 칸막이: 165×170, (215,0) — 점선 경계
- 계단: 70×140, (310, 하단+85)

### Y축: 1층 바닥=0, 1층 천장/2층 바닥=250, 2층 천장=500

## 코딩 규칙

- 커밋 메시지는 **한글**로 작성
- 기존 Blueprint3D 코드 스타일을 최대한 유지
- 새 기능 추가 시 기존 모듈 구조를 따를 것
- jQuery 의존성은 그대로 유지 (마이그레이션하지 않음)
- 가구 3D 모델은 `example/models/`의 기존 OBJ 파일만 사용
- 고정 구조물은 Three.js BoxGeometry 등 기본 도형으로 생성
