# CLAUDE.md — 프로젝트 상태 문서

## 프로젝트 개요

- **이름**: 3D Interior App (Blueprint3D 기반)
- **목표**: 기존 2D 가구 배치 시뮬레이터의 방 구조를 3D로 변환한 복층 인테리어 앱
- **베이스**: [furnishup/blueprint3d](https://github.com/furnishup/blueprint3d)
- **상태**: 🟡 셋업 단계

## 아키텍처

```
project-root/
├── src/
│   ├── core/          # 유틸리티, 설정
│   ├── items/         # 가구 아이템 타입 정의
│   ├── model/         # 데이터 모델 (방 JSON, 아이템)
│   └── three/         # 3D 렌더링, 카메라, 조명, 컨트롤
├── example/
│   ├── index.html     # 메인 진입점
│   ├── js/            # 앱 로직 (blueprint3d.js + 커스텀)
│   ├── models/        # 3D 가구 모델 (OBJ)
│   └── textures/      # 벽/바닥 텍스처
├── CLAUDE.md          # ← 이 파일
├── README.md          # 프로젝트 소개
└── .claude/
    └── instructions.md  # Claude 작업 규칙
```

## 방 구조 (핵심 데이터)

### 1층 (380cm × 510cm, 벽 높이 250cm)
| 구조물 | 크기 | 위치 | 높이 | 이동 |
|--------|------|------|------|------|
| 화장실 | 200×170 | (0,0) 좌상 | 250cm | 고정 |
| 싱크대 | 150×60 | (0,170) 좌측 | 90cm | 고정 |
| 신발장 | 100×70 | (280,0) 우상 | 120cm | 고정 |
| 계단 | 70×210 | (310,225) 우측 | 250cm | 고정 |
| 뒷문 | 100×6 | (280,504) 우하 | — | 고정 |
| 책장 | 30×100 | (0,410) 좌하 | 180cm | ⚠️이동가능 |

### 2층 (380cm × 480cm, 벽 높이 250cm)
| 구조물 | 크기 | 위치 | 높이 | 이동 |
|--------|------|------|------|------|
| 막힌공간(X) | 215×170 | (0,0) 좌상 | 250cm | 고정 |
| 칸막이 | 165×170 | (215,0) 우상 | — | 고정 |
| 계단 | 70×140 | (310, 하단+85) | 250cm | 고정 |

### 복층 Y축 배치
- 1층 바닥: y=0, 천장: y=250
- 2층 바닥: y=250, 천장: y=500

## 핵심 결정사항

| 결정 | 내용 | 이유 |
|------|------|------|
| 2D 편집 제거 | floorplanner UI를 비활성화 | 고정된 방만 사용 |
| 복층 구조 | 1층+2층 그대로 3D 변환 | 기존 2D 시뮬레이터와 동일한 구조 유지 |
| 고정 구조물 전부 유지 | 화장실, 싱크대, 계단 등 모두 3D 박스 | 원본 방 구조 충실 재현 |
| Blueprint3D 기본 모델만 | OBJ 모델 사용 | 추가 에셋 작업 불필요 |
| jQuery 유지 | 원본 의존성 그대로 | 마이그레이션 시간 절약 |
