# CLAUDE.md — 프로젝트 상태 문서

> ⚠️ **이 파일은 모든 코드 변경 전에 반드시 읽고, 변경 후에 업데이트해야 합니다.**

## 프로젝트 개요

- **이름**: 3D Interior App (Three.js 기반)
- **목표**: 기존 2D 가구 배치 시뮬레이터의 방 구조를 3D로 변환한 복층 인테리어 앱
- **베이스**: Three.js + Vite (Blueprint3D 컨셉 참고)
- **상태**: 🟢 MVP 완성

## 아키텍처

```
project-root/
├── index.html           # 메인 진입점 (UI 포함)
├── package.json         # 의존성 (three, vite)
├── vite.config.js       # Vite 빌드 설정
├── src/
│   ├── main.js          # 앱 초기화, 씬/카메라/렌더러/UI 연결
│   ├── room.js          # 1층/2층 방 구조 생성 (벽, 바닥, 고정 구조물)
│   ├── furniture.js     # 가구 정의 및 생성 (프리미티브 기반)
│   └── controls.js      # 가구 드래그/회전/삭제 인터랙션
├── dist/                # 빌드 결과물
├── CLAUDE.md            # ← 이 파일
├── README.md            # 프로젝트 소개
└── .claude/
    └── instructions.md  # Claude 작업 규칙
```

## 핵심 결정사항

| 결정 | 내용 | 이유 |
|------|------|------|
| 2D 편집 제거 | floorplanner 없음 | 고정된 방만 사용 |
| 복층 구조 | 1층+2층 그대로 3D 변환 | 기존 2D 시뮬레이터와 동일한 구조 유지 |
| 고정 구조물 전부 유지 | 화장실, 싱크대, 계단 등 모두 3D 박스 | 원본 방 구조 충실 재현 |
| Three.js 프리미티브 사용 | BoxGeometry 등으로 가구 표현 | OBJ 모델 없이도 즉시 동작 |
| Vite 빌드 | ES모듈 기반 빠른 개발 | 모던 빌드 시스템 |
| AI 모델 슬롯 예약 | 사이드바에 비활성 버튼 | 미래 확장 대비 |
