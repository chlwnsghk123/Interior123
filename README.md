# 3D Interior App

Three.js 기반의 웹 브라우저용 3D 복층 인테리어 가구 배치 앱입니다.
기존 2D 가구 배치 시뮬레이터의 방 구조를 그대로 3D로 변환했습니다.

## 주요 기능

- **3D 복층 방 렌더링**: 1층(380x510cm) + 2층(380x480cm) 복층 구조
- **고정 구조물**: 화장실, 싱크대, 계단, 신발장, 뒷문, 막힌공간, 칸막이가 3D로 렌더링
- **가구 배치**: 사이드바 카탈로그에서 클릭하여 방에 가구 배치
- **가구 조작**: 배치된 가구를 드래그(이동), 45도 회전, 삭제
- **색상 변경**: 벽, 바닥, 천장 색상을 컬러피커로 실시간 변경
- **3D 네비게이션**: OrbitControls (좌클릭 회전, 우클릭 이동, 스크롤 줌)
- **층별 보기**: 1층/2층/전체 탭으로 시점 전환

## 기술스택

- **Three.js** — 3D 렌더링
- **Vite** — 빌드 도구
- **HTML / CSS / JS** — UI

## 설치 & 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

브라우저에서 `http://localhost:3000` 접속
